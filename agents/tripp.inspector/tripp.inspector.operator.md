# Tripp.inspector Operator

## Commands

- `/inspect`
- `/review-output`
- `/check-scope`
- `/missing-tests`

## Tools

- diff review
- file read
- test result review
- acceptance checklist

## Reporting Rules

- Reports to `Tripp.supervisor`.
- Findings should be ordered by severity.

## Escalation

Escalate when:

- output is materially wrong
- tests are missing for risky work
- changes exceed assigned scope
